# CopCon

可以读作-`卡普空`

企业 - 协作 的含义

专注于为企业服务的精简化agent能力，可以快速地将agent能力集成到你的企业中

## 愿景

本项目期望满足以下需求：

1. 我在页面上想加一个AI辅助工具，来协助完成配置等信息
2. 我想让AI帮我完成一个任务
3. 我想在业务中集成一个AI客服
4. 我想用AI进行数据分析
5. 我想让整个企业的各种AI都协同合作

同时具备企业级审计能力

## 路线图

1. 核心Agent能力

一个非常精简的Agent，因为企业中的单个业务通常不需要长程执行

2. 长期记忆能力

基于向量数据库的超长期记忆能力，能够将企业中的长期业务积累增加到记忆库中，在对话中越来越聪明

3. agent间协作能力

基于A2A协议实现，让企业的不同部门之间能够跨部门协作

4. 集成化会话能力

让项目能够快速地无痛集成到你的业务平台

5. 数据访问能力

原生支持mysql、redis等常见数据源访问协议

具备强大的API集成能力，能够集成各种数据源接入
